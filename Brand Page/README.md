# Brand Page - [Live Link] (https://stalwart-pudding-a3c6e3.netlify.app/)

A responsive brand landing page for a shoe company, showcasing premium footwear with a clean and modern design.

## Description

This project is a static front-end website that serves as a brand page for a shoe company. It features a navigation bar, hero section with compelling copy, call-to-action buttons, and availability information on major e-commerce platforms.

## Features

- **Responsive Design**: Optimized for both desktop and mobile devices
- **Modern UI**: Clean design with Poppins font family
- **Navigation**: Easy navigation with menu items and login functionality
- **Hero Section**: Eye-catching headline and product showcase
- **E-commerce Integration**: Display of availability on Flipkart and Amazon
- **Call-to-Action Buttons**: Shop Now and Category buttons for user engagement

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Styling with responsive design and media queries
- **Google Fonts**: Poppins font family for modern typography

## How to View

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. The page will load with all styling and images

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Responsive Breakpoints

- Desktop: Default styles
- Mobile: Max-width 768px (stacked layout, adjusted font sizes)

## Customization

- **Colors**: Modify button colors and other styling in `css/style.css`
- **Content**: Update text and images in `index.html`
- **Fonts**: Change font family in the CSS import or style rules
- **Layout**: Adjust spacing and positioning in the CSS file
